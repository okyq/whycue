各种笔记软件对比

几乎试用了市面上所有的笔记软件，包括：notion、onenote、印象笔记、obsidian、思源笔记等。其中广受好评的有notion、obsidian、思源笔记。这些双链笔记能够在很多地方提高效率。但在试用过程中我也发现了他们或多或少的问题，挑几个比较经典的来说：

obsidian：

优点：

  1. 毫无入侵的markdown格式，迁移极其方便。
  2. 本地优先。 缺点
  3. 繁多的插件，繁杂的设置，让人很容易将注意力转移到折腾软件上，而不是注重于内容的输出。
  4. 同步的不稳定性。官方同步太贵，目前非官方的同步方式有四种：live-sync插件，remotely-save插件、syncthing软件和git同步。即使是最接近官方的live-sync，其使用体验也不太好，我出现过笔记丢失的情况。
  5. 对表格的支持不太好。（新版obsidian貌似已经改进）

思源笔记

优点

  1. 本地优先，官方提供了便捷的备份同步选项：s3。
  2. 支持pdf批注。 缺点
  3. 开发者有黑历史。
  4. 使用s3同步时，无法清除s3上无效的引用或附件，导致s3体积越来越大。几乎时笔记本体的好几倍大小。

Notion

我不使用Notion的原因只有一个：数据不在自己手里，让人感到不安心。

anytype介绍和使用教程

anytype介绍

试用这些软件之后，我开始停下来思考自己真正的需求

  1. 全平台使用
  2. 本地优先
  3. 同步方便

后面看到了anytype这款软件，试用的时候，我发现它完美契合了我所有的需求

  1. 本地优先，数据掌握在自己手中
  2. 官方提供了免费的备份服务器，容量1g。
  3. 最为惊喜的是它的同步功能，当设备处于同一内网时，数据会自动同步
  4. 导出导入方便，支持markdown。
  5. 支持kanban，日历，数据库等
